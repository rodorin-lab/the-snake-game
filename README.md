## 🎮 ゲーム概要

**クラシックスネークゲーム**のHTML5 Canvas実装。

シンプルな操作で誰でも遊べる、懐かしの蛇ゲーム。
壁にぶつからず、自分の体にぶつからず、餌を食べてスコアを稼ごう！

- 🐍 **蛇が餌を食べると成長**
- 🍎 **餌を食べるとスピードアップ**
- 💀 **壁・自分の体にぶつかるとゲームオーバー**
- 🔄 **スペースキーでリスタート**

---

## 🕹️ 遊び方

### ⌨️ キーボード操作

| キー | 動作 |
|------|------|
| ⬆️ **↑（上矢印）** | 上に移動 |
| ⬇️ **↓（下矢印）** | 下に移動 |
| ⬅️ **←（左矢印）** | 左に移動 |
| ➡️ **→（右矢印）** | 右に移動 |
| 🔄 **スペース** | ゲームオーバー後のリスタート |

### 🎯 ゲームの流れ

1. **ゲーム開始** — ページを開くと自動で開始
2. **餌を食べる** — 赤い餌を目指して蛇を動かす
3. **成長する** — 餌を食べるたびに蛇が1マス長くなる
4. **スピードアップ** — 餌を食べるたびに少し速くなる
5. **ハイスコアを目指す** — できるだけ長く生き延びろ！

### 💀 ゲームオーバー条件

- 🧱 **壁にぶつかる** — 画面端を越えると死亡
- 🐍 **自分の体にぶつかる** — 自分のしっぽに当たると死亡

---

## ⚙️ ゲームシステム

| 要素 | 詳細 |
|------|------|
| **フィールドサイズ** | 20×20マス（440×440ピクセル） |
| **初期スピード** | 7fps |
| **スピード増加** | 餌を食べるたびに +0.5fps |
| **スコア** | 餌1つ = 1ポイント |
| **初期長さ** | 2マス |

## 🚀 ローカル実行

## 🎮 ゲーム概要

クラシックスネークゲームを**全部盛り**で超絶進化！

- 🎨 **グラムちゃんカラー** — 紫銀の蛇が星空を泳ぐ
- ⚡ **3種のパワーアップ** — 無敵・スコア2倍・シールド
- 👾 **ボス戦** — スコア100で巨大スネークボス登場
- 📱 **スマホ対応** — タッチパッド + スワイプ操作
- ✨ **ネオンエフェクト** — 発光・パーティクル・画面シェイク

---

## 🕹️ 遊び方

### ⌨️ PC（キーボード）

| キー | 動作 |
|------|------|
| ⬆️ **↑ / W** | 上に移動 |
| ⬇️ **↓ / S** | 下に移動 |
| ⬅️ **← / A** | 左に移動 |
| ➡️ **→ / D** | 右に移動 |
| 🔄 **スペース** | ゲームオーバー後のリスタート |

### 📱 スマホ（タッチ）

| 操作 | 動作 |
|------|------|
| 👆 **画面をスワイプ** | スワイプ方向に蛇が移動 |
| 👆 **仮想パッド** | 画面下部の方向ボタン |

### 🎯 ゲームの流れ

```
┌─────────────────────────────┐
│  START → 餌を食べる → 成長   │
│      ↓                      │
│  パワーアップ獲得 → ボス戦   │
│      ↓                      │
│  ハイスコアを目指す！         │
└─────────────────────────────┘
```

1. **スタート** — 矢印キー or タップでゲーム開始
2. **餌を食べる** — 💗ハートを食べて蛇を成長させる
3. **パワーアップ** — ⭐💎🛡️のアイテムで超強化
4. **ボス戦** — スコア100で👾巨大スネークボス登場
5. **ハイスコア更新** — 記録は自動保存される

---

## 💎 パワーアップアイテム

| アイテム | 効果 | 持続時間 |
|----------|------|---------|
| ⭐ **スター** | **無敵化** — 壁・自分の体・ボスに当たっても死亡しない | 5秒 |
| 💎 **クリスタル** | **スコア2倍** — 餌を食べた時のスコアが2倍 | 5秒 |
| 🛡️ **シールド** | **1回だけ死亡回避** — 衝突しても1回だけ生き残る | 1回限り |

---

## 👾 ボス戦システム

| 条件 | 内容 |
|------|------|
| **発生条件** | スコア100点達成 |
| **ボスの動き** | 蛇を追いかけてくるAI |
| **ボスのHP** | 10（餌を食べると1ダメージ） |
| **報酬** | ボス撃破で **+500点** |

---

## ⚙️ ゲームシステム

| 要素 | 詳細 |
|------|------|
| **フィールド** | 20×20マス（440×440px） |
| **初期スピード** | 7fps |
| **スピード増加** | レベルアップごとに +0.5fps |
| **レベルアップ** | 50点ごとにレベルアップ |
| **スコア** | 餌 = 10pt（クリスタル時は20pt） |
| **ハイスコア** | ブラウザのlocalStorageに自動保存 |

## ✨ ビジュアルエフェクト

- 🌟 **トレイル発光** — 蛇の後ろに紫の光の軌跡
- 💥 **パーティクル爆発** — 餌を食べた時に光の粒が弾ける
- 📳 **画面シェイク** — ボス登場・ダメージ時に画面が揺れる
- 💫 **点滅エフェクト** — 無敵時に蛇がキラキラ
- 👁️ **ボス発光** — ボスが赤く脈動
